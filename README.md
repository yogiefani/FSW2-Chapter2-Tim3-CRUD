## ABOUT THIS WorkSync Admin

WorkSync adalah aplikasi yang dibangun untuk membantu mempermudah pengguna dalam mengelola pekerjaan tim. WorkSync dibangun menggunakan Express.JS, PostgreSQL, dan EJS.
